# June Kim

Building agent architectures and ephemeral systems.

## Current focus

**Agent harnesses** — [petricode](https://github.com/kimjune01/petricode) is an AGPL-3.0 coding agent built from the [Natural Framework](https://june.kim/the-natural-framework). Six roles (Perceive, Cache, Filter, Attend, Transmit, Consolidate), clean interfaces, no vendor lock-in.

**Context compaction** — [union-find-compaction](https://github.com/kimjune01/union-find-compaction) replaces flat summarization with structured compaction. Chatbots that learn between sessions instead of forgetting.

**Copyleft search** — [pageleft](https://github.com/kimjune01/pageleft) is semantic search that only indexes copyleft code. Embeddings + PageRank + license detection.

## Recent experiments

- [hangout](https://github.com/kimjune01/hangout) — Ephemeral IRC. Phoenix LiveView + raw TCP. No accounts, no history.
- [envelopay](https://github.com/kimjune01/envelopay) — Agent-to-agent payments over email via x402 SMTP headers
- [confession.website](https://github.com/kimjune01/confession.website) — Voice-first, burn-on-listen messaging with turn-based rally
- [auction-proof](https://github.com/kimjune01/auction-proof) — Lean 4 proofs for VCG auction properties
- [refactor-equivalence](https://github.com/kimjune01/refactor-equivalence) — Proving refactorings preserve semantics
